V3Project Yii2 Helpers
======================

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist v3project/yii2-helpers "*"
```

or add

```
"v3project/yii2-helpers": "*"
```
